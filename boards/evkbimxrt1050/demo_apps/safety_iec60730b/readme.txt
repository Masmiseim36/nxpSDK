Overview
========
This application demonstrates use of certified NXP Safety Library 3.0 which meets the IEC60730 class B standard.

More information
================
- All important documents (library/application user's guide, release note and certificate) are in SDK install folder\docs\safety\.
- For more information see webpage: www.nxp.com/iec60730.

Running the demo
================
- Follow Safety example user's guide \docs\safety\IEC60730BIMRT10XXSEUG.pdf.
