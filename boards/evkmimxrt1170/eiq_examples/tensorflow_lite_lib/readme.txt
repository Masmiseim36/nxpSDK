Overview
========
eIQ TensorFlow Lite library project.

Config tools limitation
This example cannot be imported into Config Tools because it is a static library project and it does not contain any MCU dependent settings.


Toolchain supported
===================
- IAR embedded Workbench  8.50.9
- Keil MDK  5.33
- GCC ARM Embedded  9.3.1
- MCUXpresso  11.3.0
