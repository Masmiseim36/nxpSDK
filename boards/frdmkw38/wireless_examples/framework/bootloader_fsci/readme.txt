Overview
========
This application is a serial Bootloader which uses FSCI commands over UART to load a new application image into the MCU's Flash.
Refer to  Kinetis FSCI Host API User's Guide documentation and host_sdk documentation on how to boot from FSCI Bootloader

Toolchain supported
===================
- IAR Embedded Workbench 8.32.3
- MCUXpresso IDE 10.3.0

Hardware requirements
=====================
- Mini/micro USB cable
- FRDM-KW38 board
