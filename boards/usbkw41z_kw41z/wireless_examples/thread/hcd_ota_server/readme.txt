Overview
========
This application represents the "Thread host controlled device ota server" of the Kinetis 
Thread Stack Demo Applications.
The host controlled device demo applications exposes a Host to Device control and management API carried
over a serial transport over USB and UART and using framing provided by FSCI (Serial Connectivity Interface) transport.
For more information please refer to the "Kinetis Thread Stack Demo Applications User's Guide.pdf" document and
"Kinetis Thread Stack Over-the-Air (OTA) Firmware Update User’s Guide.pdf".

Toolchain supported
===================
- IAR embedded Workbench 7.80.4
- MCUXpresso IDE 10.0

Hardware requirements
=====================
- USB-KW41Z board
- Personal Computer

Board settings
==============
No special board setting.

Prepare the Demo
================
1.  Connect the board to the PC host.
2.  Download the program to the target board.
3.  Press the reset button, then any other switch button on your board to begin running the demo.

Running the demo
================
Follow the instructions from the "Kinetis Thread Stack Demo Applications User's Guide.pdf" and
"Kinetis Thread Stack Over-the-Air (OTA) Firmware Update User’s Guide.pdf".
