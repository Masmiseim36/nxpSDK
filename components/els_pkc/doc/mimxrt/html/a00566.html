Construct a key derivation function (KDF) mode according to RFC2898 (PBKDF2).

Parameters
[out]pDerivationModePointer to the key derivation mode structure to be initialized.
[in]derivationAlgorithmFunction pointer to the key derivation algorithm function.
[in]macModeMac mode type, must represent an HMAC mode.
[in]iterationCountSecurity parameter

When calling mcuxClKey_derivation function with this constructor additional inputs must be provided via mcuxClKey_DerivationInput_t inputs[] parameter array. Detailed description of possible inputs is provided for each algorithm type.

status
