TRGMUX: Trigger Mux Driver

Overview

The KSDK provides driver for the Trigger Mux module of Kinetis devices.

Typical use case

TRGMUX_SetTriggerSource(TRGMUX0,kTRGMUX_Trgmux0Dmamux0,kTRGMUX_TriggerInput0,kTRGMUX_SourcePortPin);
