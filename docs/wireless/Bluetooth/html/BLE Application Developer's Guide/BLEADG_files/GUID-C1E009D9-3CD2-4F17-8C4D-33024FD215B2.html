BLE tasks configuration

Application developers are provided with four files for RTOS task initialization:

  • ble_controller_task_config.h and ble_controller_task.c for the Controller,
  • ble_host_task_config.h, and ble_host_tasks.c for the Host.

Reusing these files is recommended because they perform all the necessary RTOS-related work. The application developer should only modify the macros from *_config.h files whenever tasks need a bigger stack size or different priority settings. The new values should be overrided in the app_preinclude.h file.
