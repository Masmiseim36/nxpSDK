User interface

The IEEE 802.15.4 thread starts automatically at device startup in the role of an IEEE 802.15.4 Coordinator. It chooses the channel based on Clear Channel Assesment (CCA). No user interaction is needed.

The BLE Heart Rate Sensor user interface is identically with the standalone After flashing the board, the sensor is in idle mode (all LEDs flashing). To start advertising, press the SW4 button. When in GAP Discoverable Mode, the LED3 is flashing. When the central node connects to the peripheral, LED3 turns solid. To disconnect the node, hold the SW4 button pressed for 2-3 seconds. The node then re-enters the GAP Discoverable Mode.
