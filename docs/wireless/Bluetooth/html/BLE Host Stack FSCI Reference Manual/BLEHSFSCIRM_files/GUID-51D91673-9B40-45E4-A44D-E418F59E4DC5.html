GAP-AddDeviceToWhiteList.Request

Description

Adds a device address to the White List.

Parameters

Table 1. GAP-AddDeviceToWhiteList.Request Parameters
Parameter Size (bytes) Comments
OpGroup 1 0x47
OpCode 1 0x23
Length 1 Length in bytes of the following parameters
AddressType 1

The address type (public or random)

Possible values:

0x00: gPublic_c

0x01: gRandom_c
Address 6 The address of the white-listed device
