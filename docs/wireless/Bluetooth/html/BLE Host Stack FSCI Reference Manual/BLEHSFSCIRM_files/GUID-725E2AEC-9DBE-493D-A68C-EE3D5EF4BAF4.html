GAP-ReadRadioPowerLevel.Request

Description

Reads the power level of the controller's radio.

Parameters

Table 1. GAP-ReadRadioPowerLevel.Request Parameters
Parameter Size (bytes) Comments
OpGroup 1 0x47
OpCode 1 0x2C
Length 1 Length in bytes of the following parameters
TxReadType 1

Transmission read type

Possible values:

0x00: gTxPowerCurrentLevelInConnection_c (Reading the instantaneous TX power level in a connection)

0x01: gTxPowerMaximumLevelInConnection_c (Reading the maximum TX power level achieved during a connection)

0x02: gTxPowerLevelForAdvertising_c (Reading the TX power on the advertising channels)

0x03: gRssi_c (Reading the Received Signal Strength Indication in a connection)
DeviceId 1 The DeviceId for which the command is intended
