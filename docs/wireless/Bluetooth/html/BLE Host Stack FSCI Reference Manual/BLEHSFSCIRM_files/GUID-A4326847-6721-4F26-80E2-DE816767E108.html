L2CAP-SendLeCredit.Request

Description

Sends credits to a device when capable of receiving additional LE-frames.

Parameters

Table 1. L2CAP-SendLeCredit.Request Parameters
Parameters Size (bytes) Comments
OpGroup 1 0x41
OpCode 1 0x0D
Length 1 Length in bytes of the following parameters
DeviceId 1 The DeviceID to which credits are given
Channel Id 2 The L2CAP Channel Id assigned on the initiator
Credits 2 Number of credits to be given
Parent topic: L2CAP CB Messages
