GAP-EnableHostPrivacy.Request

Description

Enables or disables Host Privacy (automatic regeneration of a Private Address).

Parameters

Table 1. GAP-EnableHostPrivacy.Request Parameters
Parameter Size (bytes) Comments
OpGroup 1 0x47
OpCode 1 0x35
Length 1 Length in bytes of the following parameters
Enable 1 Enables or disables Host Privacy
Irk 16 x Enable Local IRK to be used for Resolvable Private Address generation
Parent topic: GAP Messages
