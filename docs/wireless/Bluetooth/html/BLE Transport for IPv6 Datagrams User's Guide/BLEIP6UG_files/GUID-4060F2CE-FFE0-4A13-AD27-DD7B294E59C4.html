Network IP Manager API

The Network IP Manager is the module that helps the application initialize and configures the IPv6 layers inside the IP stack library or their respective wrappers. It consists of 3 major APIs.

            nwkStatus_t NwkIpManager_Init(void);

This API initializes common modules for both IPv6 Router and Node devices. It initializes the 6LoBLE module and instance. It then registers the Bluetooth callbacks needed by the module to send data on the bearer and links the media interface to the IP layer. Finally it initializes the application wrappers for CoAP, Sockets, and UDP Echo. It returns a status of gNwkStatusSuccess_c or an error.

            nwkStatus_t NwkIpManager_StartNode(void);

This API configures modules for IPv6 Node devices. It registers the unicast forward procedure, opens the ND channel and sets the registration lifetime of the node. It returns a status of gNwkStatusSuccess_c or an error.

            nwkStatus_t NwkIpManager_StartRouter(void);

This API configures modules for IPv6 Router devices. It registers the unicast forward procedure, opens the ND channel, adds the initial global prefix to the interface and in the context compression table. The node also configures as a default router. It returns a status of gNwkStatusSuccess_c or an error.
