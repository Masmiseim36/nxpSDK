Starting as an IPv6 Router

As soon as the Bluetooth Stack has been configured, the Network IP Manager module is initialized. This initializes the 6LoBLE media interface and registers it to the IP layer. It also registers the current Bluetooth Link Layer to the 6lo interface. Furthermore, it initializes the UDP Echo, Sockets, and CoAP wrappers.

After initialization the node is configured to start as an IPv6 Router. This starts and configures ND on the opened interface.

            static void BleApp_Config()
{
...
   
    /* Initialize Network IP Manager module */
    NwkIpManager_Init();

    /* Start as an IPv6 Router */
    NwkIpManager_StartRouter();
}
