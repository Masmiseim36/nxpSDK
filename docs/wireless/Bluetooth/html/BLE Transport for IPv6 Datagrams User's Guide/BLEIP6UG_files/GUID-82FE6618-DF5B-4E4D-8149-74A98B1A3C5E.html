IPv6 over Bluetooth Overview

The Internet of Things assumes connectivity for all nodes, so the ideal protocol for connecting a billion devices is IPv6. The transport of IPv6 datagrams on top of a Bluetooth Host Stack is defined by the Internet Engineering Task Force (IETF) and depicted in RFC 7668.
