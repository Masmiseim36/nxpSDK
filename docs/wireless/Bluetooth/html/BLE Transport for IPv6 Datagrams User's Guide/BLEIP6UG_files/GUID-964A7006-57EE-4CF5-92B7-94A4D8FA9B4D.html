Creating the network

To start the scanning of devices, press the SW4 button on the router board. GAP Discoverable Mode can be entered when we press the SW4 button on the node board. The host connects with the board after it sees it advertise the IPSP service and connects to it. A solid LED3 indicates a successful connection between the two devices. This is also visible in the console, as shown below.

The IPv6 Node shows connected and has established a credit-based channel.
