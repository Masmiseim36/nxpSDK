Multicast group

To verify the multicast groups that the devices have joined, enter mcastgroup show.

The IPv6 Node will join the all-nodes multicast address.

            BLE IPv6 Node>mcastgroup show

Interface 0:
Multicast Group: ff02::1

The IPv6 Router will join the all-routers multicast address and the all-nodes multicast address.

            BLE IPv6 Router>mcastgroup show

Interface 0:
Multicast Group: ff02::2
Multicast Group: ff02::1
