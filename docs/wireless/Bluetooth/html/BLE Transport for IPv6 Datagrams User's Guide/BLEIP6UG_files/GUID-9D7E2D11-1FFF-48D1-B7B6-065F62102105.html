User interface

After flashing the boards, the devices are in idle mode (all LEDs flashing).

Press the SW4 button to start scanning on the IPv6 Router. When in GAP Limited Discovery Procedure, LED3 starts flashing. To start advertising on the IPv6 Node, press SW4. When in GAP Discoverable Mode, LED3 starts flashing.

When the nodes are connected, LED3 turns solid. To disconnect the nodes, hold the SW4 button pressed for 2-3 seconds on the IPv6 Node. The IPv6 Node reenters GAP Discoverable Mode.
