Topologies

IPv6 over Bluetooth supports only a hub-and-spokes topology, as shown in Figure 2. Therefore, two types of devices exist:

  • 6LBR – 6LoWPAN Border Router which is an IPSP Router and a GAP Central.
  • 6LN – 6LoWPAN Node which is an IPSP Node and a GAP Peripheral.
Figure 1. Isolated Bluetooth IPv6 Network

All 6LN devices and the Bluetooth interface of the 6LBR are in the same IPv6 subnet. Optionally, the 6LBR may have an additional interface over Ethernet to communicate with the Internet, as shown in Figure 3.
