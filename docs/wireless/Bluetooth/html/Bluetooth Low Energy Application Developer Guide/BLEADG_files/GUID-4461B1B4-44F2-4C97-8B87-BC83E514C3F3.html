GAP generic event

The GAP Generic Event is triggered by the Host Stack and sent to the application via the generic callback. Before any application-specific interactions, the Connection Manager callback is called to handle common application events, such as device address storage. 

void BleApp_GenericCallback ( gapGenericEvent_t * pGenericEvent)
{
    /* Call Bluetooth Low Energy Conn Manager */
    BleConnManager_GenericEvent(pGenericEvent);

    switch (pGenericEvent-> eventType )
    {
        ...
    }
}
