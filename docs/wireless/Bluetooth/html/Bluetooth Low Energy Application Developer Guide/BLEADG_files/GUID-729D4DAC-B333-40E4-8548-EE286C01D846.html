Deep sleep modes for QN908X MCU-based platforms

QN908x has only two deep sleep modes called power down 0 (PD0) and power down 1 (PD1). See “UM11023 QN908x User Manual” for details. For both modes the required RAM memory blocks are powered up (the user can configure the blocks that stay powered up and powered down during deep sleep). CPU registers values can be retained during deep sleep (the user can disable this feature).

See Low power library for QN908X MCU based platforms section of Connectivity Framework Reference Manual for the list of supported deep sleep modes and their description.
