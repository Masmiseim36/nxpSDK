Hybrid (Dual-Mode) Bluetooth® Low Energy and IEEE® 802.15.4 Applications (KW41 Platform)

This section describes how to add IEEE 802.15.4 functionality to an existing Bluetooth Low Energy application in order to create a dual mode application.

KW3x, QN908X, and K32W061 MCU-based platforms do not support 802.15.4, so this chapter about hybrid applications does not apply to these platforms.
