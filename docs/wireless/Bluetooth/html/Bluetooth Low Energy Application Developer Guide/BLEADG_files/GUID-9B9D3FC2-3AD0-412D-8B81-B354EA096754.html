Periodic Advertising (KW37)

Periodic channels are used for periodic broadcast between unconnected devices. A periodic channel is represented by a channel map and a set of hopping and timing parameters.

The set of channels is represented by the 37 data channels. A packet sent by an advertiser can have also have a payload of up to 255 bytes and it can be sent on any LE PHY.
