Using low-power when scanning

Scanning requires the Bluetooth Low Energy Link Layer to be in running mode during the whole procedure. The device enters sleep after the scanning is finished or remain active if a suitable device is found. So, the deep sleep mode 1 is the best candidate for this use case.

To allow the device to enter deep sleep mode 1, call PWR_ChangeDeepSleepMode before starting a scan. If the scanner performs a low-duty cycle scan (the scan window value is smaller than the scan interval value), the system will enter a low-power mode when the scanning window finishes.

PWR_ChangeDeepSleepMode(1);
App_StartScanning();
