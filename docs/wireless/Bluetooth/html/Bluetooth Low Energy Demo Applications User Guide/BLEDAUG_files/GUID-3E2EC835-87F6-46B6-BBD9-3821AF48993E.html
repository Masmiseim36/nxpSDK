Supported platforms

The Temperature Sensor and Collector applications are supported by the following platforms:

  • FRDM-KW41Z
  • USB-KW41Z
  • QN9080CDK
  • FRDM-KW36
    Note: On FRDM-KW36 and FRDM-KW38, deep-sleep modes 5 and 8 are used by default.
