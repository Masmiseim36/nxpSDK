User interface

After flashing the board, the beacon application starts advertising. When in GAP Discoverable Mode, CONNLED is flashing. Hold the ADVSW button pressed for 2-3 seconds anytime to stop advertising. In idle mode, all LEDs are flashing. To restart advertising, press the ADVSW button.
