Freedom FRDM-KW36 platform

FRDM-KW36 Freedom platform is based on the KW36 SoC which incorporates an Arm® Cortex®-M0+ core and enables Bluetooth Low Energy and Generic FSK. It can be configured to operate at various frequencies, up to 48 MHz and it has 256 KB of on-chip Flash and 64 KB of SRAM. The Bluetooth LE transceiver supports up to 8 concurrent connections in any master/slave combination. A figure representing the platform is shown below. For detailed information about the platform, see the appropriate board user’s guide.

The platform features a composite USB device called OpenSDA which serves as debugger interface and as a USB-to-serial converter via a virtual COM port application.
