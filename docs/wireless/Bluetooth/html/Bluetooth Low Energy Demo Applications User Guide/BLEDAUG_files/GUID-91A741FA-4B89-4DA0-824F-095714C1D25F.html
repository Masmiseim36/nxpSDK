User interface

After flashing the board, the sensor is put in deep sleep (all LEDs are off). Also, any attached debugger will lose its connection. The application default configuration enables low power, which disables LED support. The user can manually change this and enable LED support, otherwise all subsequent LED behavior references shall be ignored. To wake up and start advertising, press the ADVSW button. When in GAP Discoverable Mode, the CONNLED is on. When the central node connects to the peripheral, the CONNLED remains solid. The heart rate measurement values are generated randomly in the interval 40 to 200 bpm. The heart rate value format is set to 8 bits and the contact status is on.
