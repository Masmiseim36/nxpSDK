User interface

After flashing the board, both nodes enter cPWR_DeepSleepMode deep-sleep mode (3- LLS or 5 - VLLS). To flash the board in case the sensor is put in deep sleep, press WAKESW or RESET. The application default configuration enables low power which disables LED support. The user can manually change this and enable LED support, else all subsequent LED behavior references are ignored. All LEDs are off. When the node is awake and communicating, CONNLED is on. To wake up the node press the WAKESW button. For FRDM-KW38 board, the default enabled low-power modes are 1 and 5. This means that the MCU is put into VLLS2 or VLLS3 stop mode. In this mode, the LED module operation is not implemented.
