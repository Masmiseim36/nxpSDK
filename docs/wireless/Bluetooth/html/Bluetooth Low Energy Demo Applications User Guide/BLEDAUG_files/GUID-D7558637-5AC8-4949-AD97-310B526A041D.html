Secured OTAP for multicore

FRDM-K32W042 is a multicore platform equipped with two cores: an Arm Cortex-M4 core and an Arm Cortex-M0+ core. The process for performing OTA updates on this platform differs from single core platforms:

  • Selective OTAP can be performed. A single OTA update can include only the M4 image, only the M0+ image or both.
  • The image files (*.bin or *.srec) are encrypted and signed beforehand, resulting in a *.sbin file.
