GAP-SaveKeys.Request

Description

Save the bond keys to an NVM index.

Parameters
Table 1. GAP-SaveKeys.Request
Parameter Size (bytes) Comments
OpGroup 1 0x48
OpCode 1 0x42
Length 1 Length in bytes of the following parameters

The SMP keys distributed by the peer

Structure type parameter. See detailed table below for parameter structure.
LeSc 1 LE SC connections pairing
Authenticated 1 Peer was authenticated during pairing
