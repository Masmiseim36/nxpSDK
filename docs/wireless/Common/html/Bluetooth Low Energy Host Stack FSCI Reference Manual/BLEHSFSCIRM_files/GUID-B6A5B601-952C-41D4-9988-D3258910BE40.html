GAP-Authorize.Request

Description

Authorizes a peer for a certain attribute in the database.

Parameters

Table 1. GAP-Authorize.Request Parameters
Parameter Size (bytes) Comments
OpGroup 1 0x48
OpCode 1 0x07
Length 1 Length in bytes of the following parameters
DeviceId 1 The peer being authorized
Handle 2 The attribute handle
Access 1

GATT Database access type (the type of access granted - read or write)

Possible values:

0x00: gRead_c (Attempting to read the attribute)

0x01: gWrite_c (Attempting to write the attribute)
