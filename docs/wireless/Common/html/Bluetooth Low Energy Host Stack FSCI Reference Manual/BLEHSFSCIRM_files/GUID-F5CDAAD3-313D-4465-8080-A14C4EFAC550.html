L2CAPCB-CancelConnection.Request

Description

Termination of an L2CAP channel.

Parameters

Table 1. L2CAPCB-CancelConnection.Request Parameters
Parameter Size (bytes) Comments
OpGroup 1 0x42
OpCode 1 0x06
Length 1 Length in bytes of the following parameters
LePsm 2 Bluetooth SIG or Vendor LE_PSM
DeviceId 1 The DeviceId for which the command is intended
RefuseReason 2

Refuse reason

Possible values:

0x0000: gSuccessful_c

0x0002: gLePsmNotSupported_c

0x0004: gNoResourcesAvailable_c

0x0005: gInsufficientAuthentication_c

0x0006: gInsufficientAuthorization_c

0x0007: gInsufficientEncryptionKeySize_c

0x0008: gInsufficientEncryption_c

0x0009: gInvalidSourceCid_c

0x000A: gSourceCidAlreadyAllocated_c

0x0100: gCommandRejected_c

0xFFFE: gResponseTimeout_c
