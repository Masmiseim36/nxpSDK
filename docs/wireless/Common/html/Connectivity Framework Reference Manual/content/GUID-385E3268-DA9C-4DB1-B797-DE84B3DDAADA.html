Overview

The RNG module is part of the framework used for random number generation. It uses hardware RNG peripherals, IEEE 802.15.4 PHY RNG module, and a software pseudo-random number generation algorithm. If no hardware acceleration is present, the RNG module uses a software algorithm. The initial seed for this algorithm represents the device unique ID (SIM_UID registers) by default. Use the 802.15.4 PHY RNG for the initial seed by setting the gRNG_UsePhyRngForInitialSeed_d define to 1.
