Overview

The behavior is similar for all operating systems, except for the allocation procedure. For bare metal, the semaphores are allocated within the OS Abstraction layer, while FreeRTOS OS and µC-OS/II OS allocates them internally. For bare metal there is also no context switch when the semaphore can’t be taken, so the function status must be checked.

The osNumberOfSemaphores define controls the maximum number of semaphores permitted.
