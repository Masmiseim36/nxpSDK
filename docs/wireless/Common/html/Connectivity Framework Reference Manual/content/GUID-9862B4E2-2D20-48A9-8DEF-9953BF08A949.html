Overview

The module enables control of up to four LEDs using a low-level driver. It offers a high-level API for various operation modes:

  • Flashing
  • Serial flashing
  • Blip
  • Solid on
  • Solid off
  • Toggle
  • RGB dimming, if an RGB LED is available and connected to PWM channels.

The flash and blip features use a timer from the Timersâ€™ Manager module. If the RGB dimming is enabled, another separate timer is allocated from Timer Manager module.
