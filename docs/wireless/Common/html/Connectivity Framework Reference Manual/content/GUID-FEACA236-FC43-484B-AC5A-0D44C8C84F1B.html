Overview

The module enables controlling of up to four switches using a low-level driver. It offers a high-level API for various operation modes:

  • Press only
  • Short/long press
  • Press/hold/release

The keyboard event is received by the user code by executing a user-implemented callback in an interrupt context. The keyboard uses a timer from the Timersâ€™ Manager module for debouncing, short/long press, and hold detection.
