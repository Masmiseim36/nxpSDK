Enabling CoAP observe

CoAP Observe is by default disabled in all application projects. It can be enabled from all application configuration files, located at:

\middleware\wireless\nwk_ip_1.2.8\examples\<thread_application>\config\config.h

/* Enable CoAP Observe Client */
#define COAP_OBSERVE_CLIENT       1

/* Enable CoAP Observe Server */
#define COAP_OBSERVE_SERVER       1

The CoAP Observe Client and Server are started from the Shell command line. For projects that do not support Shell, call APP_ObserveStartClient() to initialize the Observe Client and/or APP_ObserveStartServer() to initialize the Observe Server. For more information about the interface functions, see the section below.
