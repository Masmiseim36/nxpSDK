Joining a network

For an application to initiate and join an existing Thread network the THR_Join API is called. The API supports initiating joining with in-band commissioning or with pre-commissioned settings matching an out-of-band commissioning configuration. The commissioning mode can be set at compile time using the THR_DEV_IS_OUT_OF_BAND_CONFIGURED parameter.
