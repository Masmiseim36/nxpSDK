MCUXpresso IDE project options configuration

The configuration settings are primarily contained in the MCUXpresso Project Settings panel and the *.h header files contained in \middleware\wireless\nwk_ip_1.2.4\examples\common and \middleware\wireless\nwk_ip_1.2.4\examples\<application>\config file groups within a project.

To see the configuration options set within the MCUXpresso Project Settings panel, right-click on the application project and select the Properties option.

The figure below shows the global Preprocessor options. These include the following:

  • Setting the file at \middleware\wireless\nwk_ip_1.2.4\examples\<application>\config\ config.h as a Preinclude file which is included first by default whenever a source module compilation is invoked
  • Preprocessor Defined symbols which consist in the main global settings for the system and stack for the application project
