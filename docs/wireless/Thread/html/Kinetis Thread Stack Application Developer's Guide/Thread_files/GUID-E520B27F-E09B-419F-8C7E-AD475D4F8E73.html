CoAP observe

CoAP Observe module is an extension of the CoAP core protocol. The source files are located at \middleware\wireless\nwk_ip_1.2.4\examples\common\app_coap_observe.c.

A test application is delivered with the Observe module. The application source files are located at \middleware\wireless\nwk_ip_1.2.4\examples\common\app_observe_demo.c. The application needs the Shell module to be enabled.

CoAP Observe works on a client and server model. The server is the authority for a specific resource/resources that changes its state/value. The client subscribes to a server to receive updates for a specific resource.
