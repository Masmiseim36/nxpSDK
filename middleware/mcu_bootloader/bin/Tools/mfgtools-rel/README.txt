README for MfgTool
===================

1. To run MfgTool2.exe, please rename cfg_MXRT1xxx.ini to cfg.ini
2. Please create new foldler named blhost, at the same layer with mfgtools-rel, and create a new folder named win under blhost. Download the blhost_2.x.x package from www.nxp.com/mcuboot, extract the blhost.exe from the package and put it under blhost/win.
  Directory layout of mfgtool is:
    Tools
      |--blhost
      |    |--win
      |        |--blhost.exe
      |
      |--mfgtools-rel
      |    |--MfgTool2.exe
      |    |--cfg.ini
      |    |--Profiles
      |    |--...
      |
      |--...
