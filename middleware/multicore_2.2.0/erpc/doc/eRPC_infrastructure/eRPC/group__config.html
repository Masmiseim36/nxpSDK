#define ERPC_DEFAULT_BUFFER_SIZE

Size of buffers allocated by BasicMessageBufferFactory in setup functions.

Uncomment to change the size of buffers allocated by BasicMessageBufferFactory in the client and server setup functions (Client Setup and Server Setup). The default size is 256.
